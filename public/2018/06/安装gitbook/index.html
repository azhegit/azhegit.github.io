安装GitBook

2018 年 六月 8 号 发布在 技术, 开发环境
  1. 查看本机node环境：node -v

localhost:~ azhe$ node -v v10.5.0

  1. 安装gitbook:sudo npm install gitbook-cli -g
  2. 查看gitbook版本:gitbook -V

localhost:~ azhe$ gitbook -V CLI version: 2.3.2 GitBook version: 3.2.3

  1. 初始化gitbook目录:gitbook init
  2. 启动:gitbook serve -p 8080
